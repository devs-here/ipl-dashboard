# 🏏 IPL Analysis Dashboard

A data analysis dashboard analyzing IPL cricket data from 2008 to 2024.

## 📊 What's Inside
- Most wins by team
- Top 10 run scorers
- Top 10 wicket takers
- Toss decision analysis
- Player of the match awards
- Total runs per season
- Top venues
- Runs vs boundaries scatter plot
- Top 10 boundary hitters
- Highest team total in a single innings
- Team wins per season heatmap

## 🛠️ Built With
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## 📁 Dataset
IPL Complete Dataset from Kaggle (2008-2024)
